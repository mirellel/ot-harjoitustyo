# Word.py
Sovelluksessa käyttäjä voi pelata Wordle-kaltaista sanaarvauspeliä. Sovellus ja peli ovat englanninkielisiä.
Sovellukseen on mahdollista rekisteröityä käyttäjänä, jolloin voi nähdä mm. omat pelitilastonsa.
